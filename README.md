# my-first-repo 实训项目
git仓库
## 功能特性
- git仓库搭建
- 远程仓库同步
- 支持多人协作
- 随时回溯，有历史记录留存
## 安装步骤
1. 安装git
2. 使用`git clone`命令拉取本仓库
3. 本地编辑、新增项目文件
4. 执行提交命令同步到远程仓库
## 代码示例
`git add .`
`git commit -m ''`
`git switch main`
